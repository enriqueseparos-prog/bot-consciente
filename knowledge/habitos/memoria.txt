**Definición de la Memoria**

La memoria se refiere a la capacidad del cerebro para almacenar y recuperar información a partir de experiencias pasadas. Hay diferentes tipos de memoria, incluyendo:

- Memoria a corto plazo: capacidad para recordar información durante un período corto de tiempo (milisegundos a minutos)
- Memoria a largo plazo: capacidad para recordar información durante un período largo de tiempo (horas, días, semanas, meses, años)
- Memoria sensorial: capacidad para recordar sensaciones y experiencias sensoriales
- Memoria de trabajo: capacidad para recordar información que se utiliza en actividades cognitivas

**Tipos de Memoria**

- Memoria episódica: capacidad para recordar eventos y experiencias personales
- Memoria semántica: capacidad para recordar conceptos y relaciones entre ellos
- Memoria procedimental: capacidad para recordar habilidades y procedimientos
